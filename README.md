# Patient No-Show Prediction and Healthcare Analytics Dashboard Using Python, PostgreSQL, and Power BI

## Project Overview

This project focuses on analyzing hospital patient appointment data to identify patterns behind patient no-shows. The system helps hospitals improve appointment management and reduce missed appointments using data analytics and visualization techniques.

The project was developed using:

* Python for data preprocessing
* PostgreSQL for database management
* Power BI for dashboard visualization

---

# Objectives

* Analyze patient appointment attendance behavior
* Identify factors affecting patient no-shows
* Visualize healthcare insights using dashboards
* Improve hospital scheduling efficiency
* Support better healthcare decision-making

---

# Technologies Used

* Python
* Pandas
* PostgreSQL
* SQL
* Power BI

---

# Dataset Features

The dataset includes:

* Patient Age
* Scholarship Status
* Hypertension
* Diabetes
* Alcoholism
* SMS Reminder Status
* Waiting Time
* Attendance Prediction

# Project Workflow

Using Python and Pandas:

* Removed missing values
